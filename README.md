#Descripción y aprendizajes
Es el primer proyecto de la asignatura que nos ha servido para empezar a familiarizarnos con java y con la Programación
Orientada a Objetos (POO). Hemos aprendido a crear clases con variables, constructores, getters, setters, métodos, etc.
Nos hemos familiarizado con la manera en la que interactuan unas clases con otras, como crear e implementar clases 
abstractas e interfaces y como crear tests de cobertura para asegurar el correcto funcionamiento de las clases
Además no solo nos ha servido para programar con java sino que también hemos experimentado con Git y como crear
repositorios para poder guardar nuestro trabajo y compartirlo con otros compañeros, lo cual nos ha servido para
programar en equipo que al fin y al cabo es como se trabaja en el mundo real y en definitiva a adquirir conocimientos y 
habilidades que nos servirán para nuestro futuro académico y profesional.
